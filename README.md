# sai2-capture
sai2可用的窗口捕捉，绘画过程录制

1.打开sai2的 窗口-分离操作面板-分离导航器面板   

2.在 帮助-设置-首选项 里设置合适的导航器最大大小

3.文件会保存在exe文件所在目录的output_frames文件夹里

4.选择你的用户界面缩放比例

5.点击生成视频，选择文件夹，即可将文件夹内图片按编号合成成视频

6.生成完视频后请务必手动酌情删除图片，否则图片会占用大量磁盘空间

生成视频的时候可能会卡住，但是实际上还是在运行的
