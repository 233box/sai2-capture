# 🎨 SAI2 窗口捕捉与绘画步骤录制工具

---

## 📌 简介

本工具专为 [PaintTool SAI 2]用户设计，能够轻松（并非轻松）捕捉操作界面并录制你的绘画步骤，生成视频。

readme格式由ai加工

大部分代码由ai编写

---

## 🛠️ 使用步骤

1. **打开 SAI2 的分离导航器面板**  
   路径：`窗口 → 分离操作面板 → 分离导航器面板`

2. **设置导航器大小**  
   进入 `帮助 → 设置 → 首选项`，调整“导航器最大大小”以适应你的屏幕分辨率。

3. **图像保存路径**  
   所有捕获的帧将自动保存在程序所在目录下的 `output_frames` 文件夹中。

4. **选择界面缩放比例**  
   根据你的显示设置选择合适的 UI 缩放比例，以确保捕捉效果最佳。

5. **开始**  
   点击开始即可开始录制
